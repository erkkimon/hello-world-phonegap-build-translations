hello-world-phonegap-build-globalization
========================================

Hello World of PhoneGap Globalization plugin to be built with PhoneGap Build Cloud.

I can't fix the bug, but if someone can do something, something would be appreciated. :) More information can be found here: https://issues.apache.org/jira/browse/CB-4474
